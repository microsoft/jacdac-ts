## Format

Tokens are white-space separated,
* ``%`` - number from args[]
* ``#`` - color from args[]
* ``#0123ff`` - color
* ``123`` - number
* commands
    * ``setall``
    * ``fade``
    * ``fadehsv``
    * ``rotfwd``
    * ``rotback``
    * ``show``
    * ``wait``
    * ``range``
    * ``mode``
    * ``tmpmode``
    * ``setone``
    * ``mult``
