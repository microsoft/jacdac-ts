## Format

Tokens are white-space separated,
* ``%`` - number from args[]
* ``#`` - color from args[]
* ``#0123ff``` - color
* ``123``` - number
* commands: set, fade, fadehsv, rotfwd, rotback, pause
    * setall": return 0xD0
    * fade": return 0xD1
    * fadehsv": return 0xD2
    * rotfwd": return 0xD3
    * rotback": return 0xD4
    * show":
    * wait": return 0xD5
    * range": return 0xD6
    * mode": return 0xD7
    * tmpmode": return 0xD8
    * setone": return 0xCF
    * mult": return 0x100
